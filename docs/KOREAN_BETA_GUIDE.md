# PawMemo 한국어 베타 테스트 가이드

이 문서는 영어 UI로 제공되는 PawMemo를 한국어 베타 테스터가 테스트할 때 참고할 수 있는 안내서입니다. 실제 API 키, 비밀번호, Supabase 키, OpenAI 키, 테스트 계정 정보는 이 문서에 넣지 않습니다.

## 테스트 목적

PawMemo가 반려동물의 일상 기록을 쉽고 따뜻하게 남길 수 있는 서비스인지 확인합니다.

특히 아래를 보고 싶습니다.

- 처음 접속했을 때 어떤 서비스인지 이해되는지
- 반려동물 프로필을 만들고 기록을 남기는 흐름이 쉬운지
- 사진 기록, 태그, Timeline, Reports를 자연스럽게 찾을 수 있는지
- AI 태그 추천과 Weekly Paw Letter가 부담 없이 도움이 되는지
- Vet-ready Summary가 진단처럼 보이지 않고, 병원 상담 전 기록 정리용으로 안전하게 느껴지는지

## 테스트 시작 전 안내

- 현재 PawMemo 앱 UI는 영어입니다.
- 테스트 중 실제 민감한 의료 정보나 긴급 상황 기록은 입력하지 않는 것을 권장합니다.
- AI 기능은 선택 사항입니다. 꼭 사용하지 않아도 됩니다.
- Vet-ready Summary는 기록을 정리해주는 기능이며 진단이 아닙니다.
- 문제가 생기면 화면, 사용한 기기, 어떤 단계였는지 함께 알려주세요.

## 테스트 흐름

아래 순서대로 천천히 테스트해 주세요.

1. `https://pawmemo.vercel.app`에 접속합니다.
2. 첫 화면을 보고 어떤 서비스라고 느껴지는지 기억해 둡니다.
3. 계정을 새로 만들거나 로그인합니다.
4. 반려동물 프로필을 만듭니다.
5. `Journal` 화면에서 짧은 텍스트 기록을 남깁니다.
6. 사진이 포함된 기록을 하나 남깁니다.
7. 기록을 남길 때 수동 태그를 하나 선택해 봅니다.
8. `Suggest tags`를 눌러 AI 태그 추천을 한 번만 사용해 봅니다.
9. `Timeline`에서 방금 남긴 기록이 보이는지 확인합니다.
10. `Reports`로 이동합니다.
11. `Weekly Paw Letter`를 생성해 봅니다.
12. `Vet-ready Summary`를 생성해 봅니다.
13. `Saved reports`에서 저장된 리포트를 확인합니다.
14. `Copy report`를 눌러 리포트 복사를 시도해 봅니다.
15. `Print / Save as PDF`를 눌러 브라우저 인쇄 또는 PDF 저장 흐름을 확인해 봅니다.
16. `Settings` 화면을 확인합니다.
17. 시간이 있으면 로그아웃 후 다시 로그인해서 기록이 유지되는지 확인합니다.

## 영어 UI 용어표

| 영어 UI | 한국어 의미 |
| --- | --- |
| Journal | 기록을 남기는 메인 화면 |
| Timeline | 저장한 기록을 시간순으로 보는 화면 |
| Reports | 편지와 요약 리포트를 만드는 화면 |
| Settings | 계정과 앱 상태를 확인하는 설정 화면 |
| Quick Entry | 빠르게 기록을 남기는 입력 영역 |
| Add photo | 사진 추가 |
| Suggest tags | AI 태그 추천 |
| Weekly Paw Letter | 최근 기록으로 만드는 따뜻한 주간 편지 |
| Vet-ready Summary | 병원 상담에 가져갈 수 있도록 기록을 정리한 요약 |
| Saved reports | 저장된 편지와 요약 목록 |
| Copy report | 리포트 내용 복사 |
| Delete report | 저장된 리포트 삭제 |
| Print / Save as PDF | 브라우저 인쇄 또는 PDF 저장 |

## 피드백 질문

테스트 후 아래 질문에 답해 주세요.

- 처음 봤을 때 어떤 서비스인지 이해됐나요?
- 회원가입/로그인이 어렵지 않았나요?
- 반려동물 프로필 생성이 자연스러웠나요?
- 기록을 남기는 흐름이 쉬웠나요?
- 사진 기록을 남기는 과정이 이해됐나요?
- Timeline, Reports, Settings로 이동하는 방법이 명확했나요?
- AI 태그 추천이 도움이 됐나요?
- Weekly Paw Letter가 감성적으로 가치 있었나요?
- Vet-ready Summary가 병원 상담용으로 실용적으로 느껴졌나요?
- Vet-ready Summary가 진단이나 의료 조언처럼 느껴진 부분은 없었나요?
- 영어 UI 때문에 막히는 부분이 있었나요?
- 한국어 UI가 꼭 필요하다고 느낀 화면은 어디였나요?
- 어떤 부분이 가장 헷갈렸나요?
- 어떤 부분이 가장 마음에 들었나요?
- 다시 사용할 이유가 있었나요?

## 알려진 제한 사항

베타 테스트 전에 아래 내용을 알고 진행해 주세요.

- 현재 앱 UI는 영어입니다.
- 결제는 아직 지원하지 않습니다.
- `Pricing`은 미리보기입니다.
- `Pet profile photo upload`는 아직 없습니다.
- `Account deletion`은 아직 없습니다.
- `Export all memories`는 아직 없습니다.
- AI suggestions는 선택 사항입니다.
- Vet-ready Summary는 기록 정리용이며 진단이 아닙니다.
- 저장된 리포트는 브라우저의 인쇄 기능으로 PDF 저장을 시도할 수 있지만, 별도의 PDF 생성 기능은 아직 없습니다.

## 한국어 UI 관련 다음 단계

이번 단계에서는 앱 UI를 한국어로 바꾸지 않습니다.

한국어 베타 테스터가 영어 UI 때문에 자주 막힌다면 다음 단계는 `Phase 13C: lightweight Korean/English language support`가 됩니다. 그 단계에서는 핵심 화면부터 한국어/영어 지원 방식을 작게 검토합니다.
