# PawMemo

PawMemo is a gentle pet memory journal for saving quick notes, photos, care signals, and meaningful moments in one private space.

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS
- React
- lucide-react

## Current Status

Phase 1 static UI with mock data.

The app currently includes static, mobile-first screens based on mock data only. No backend, authentication, storage, AI, payment, or production PWA flow is connected yet.

## Getting Started

```bash
npm install
npm run dev
```

Open `http://localhost:3000` in your browser.

## Main Routes

- `/` - Landing Page
- `/onboarding` - Onboarding
- `/app` - Home / Quick Entry
- `/app/timeline` - Timeline with Search
- `/app/mosaic` - Memory Mosaic
- `/app/reports` - Reports Hub
- `/app/reports/weekly` - Weekly Paw Letter
- `/app/reports/vet-summary` - Vet-ready Summary
- `/pricing` - Pricing
- `/app/settings` - Settings

## Not Implemented Yet

- Supabase Auth
- Supabase Storage
- AI Edge Functions
- Lemon Squeezy payments
- PWA install flow
